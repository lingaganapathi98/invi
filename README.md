# Wedding Invitation React Website

A minimal and elegant wedding invitation website built with React and Vite.

## How to use

1. Extract the zip.
2. Open the project folder in your terminal.
3. Install dependencies:

```bash
npm install
```

4. Start the development server:

```bash
npm run dev
```

5. Open the local URL shown in the terminal.

## What to edit

Open `src/App.jsx` and replace the placeholder values inside `invitationData`:

- Groom name
- Bride name
- Groom photo path
- Bride photo path
- Groom parents
- Bride parents
- Date
- Time
- Venue name
- Address
- Google Maps URL

## Photos

Replace these files in the `public` folder with your real images:

- `public/groom.jpg`
- `public/bride.jpg`

Keep the same file names, or update the paths in `src/App.jsx`.

## Build for production

```bash
npm run build
```

The final static files will be generated in the `dist` folder.
